## 拍照功能现状（最近整合完成、tsc/vitest 全绿）
- **主路径 = LLM 通用拍照**：人物在回复里自然输出 `[[PHOTO|带人=…|在家=…|画面=…]]` 标记 → `photo-intent.ts` 解析剥离 → `message-handler.ts` 异步发图 → `selfie-provider.generatePersonaPhoto`（带人/在家/外面三态，附对应参考图）。
- **兜底 = 规则明确指令**：用户明确说「发自拍 / 拍家里」→ `decideSelfieOpportunity` 必发、破冷却。正则已放宽到识别口语（`selfie-commands.ts` 自拍 / `selfie-decision.ts` 环境）。
- **三段式发送**：文字「预告」（prompt 约束只预告、不用文字演照片）→（网页生图一两分钟）→ 实拍 → 「交付语」（`selfie-delivery.ts`，selfie/environment/scene 三类话术池、避免连续重复）。
- **区域库**：`PERSONA_SELFIE_HOME_DIR` 放 bedroom/kitchen/livingroom/study/bathroom/entry/balcony.png，`resolveHomeRegionRef` 按场景关键词+作息选（深夜→卧室）。
- **门控**：冷却（3h / 每日 2 张，`selfie-cooldown.ts`）+ 作息（深夜等静默时段不主动注入拍照标记）。
- 开关 `PERSONA_SELFIE_ENABLED`（默认关）。生图经本机 `chatgpt-project-prompt-pusher`（.NET `--generate` 推网页版 ChatGPT，独占 Chrome profile `google-ai-persistent-profile`，仅 SFW）。
